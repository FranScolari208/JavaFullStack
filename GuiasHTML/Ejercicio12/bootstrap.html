Programacion

La programación es el proceso utilizado para idear y ordenar las acciones necesarias para realizar un proyecto. En la actualidad, la noción de programación se encuentra muy asociada a la creación de aplicaciones de informática y videojuegos.

Programando con Java

Java es un lenguaje de programación y una plataforma informática que fue comercializada por primera vez en 1995 por Sun Microsystems. Hay muchas aplicaciones y sitios web que no funcionarán, probablemente, a menos que tengan Java instalado, y cada día se crean más.
